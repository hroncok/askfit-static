fakultní Jenkins + bitbucket

Ahoj, nastavoval jste někdo fakultní jenkins pro repozitář na bitbucket ?

Linky vám nepošlu, nemůžu je vložit, sry...

Funguje vám to? Jsou nějaké kroky, které je nutno přesně provést?

Po 50+ neúspěšných pokusech a tunách googlingu už mě to nebaví. Než se obrátim na někoho z kantorů, tak jsem se chtěl zeptat tady...

Co to dělá:

ERROR: Error fetching remote repo 'origin' nebo Could not read from remote repository - nejspíš chyba autorizace, kterou jsem snad už vyřešil

(představujte si krásné URL, damn you, antispam protection)

Co jsem zkusil: napojit na fit gitlab nebo git project, taky bez úspěchu, Ani podle návodu na rozvoj . fit ...

Jak to mám teď:

Jenkins:

ssh připojení na [email protected](tečka)org:teamname/project(tečka)git

"**/master" branch

repo browser bitbucketweb, (url bitbucket httpsdomena):teamname/project.git

Trigger build remotely a náhodnej token, co jsem si vyťukal na klávesnici

Build: Invoke ant, a relativní cesta k němu

Na bitbucketu máme týmovej projekt:

nastavenej deployment key SSH klíč z návodu

nastavenej jenkins hook s APIkey mého jenkins účtu

endpoint: (https)kubisada:(předtim vyťukanej random token)@(fit jenkins url)

project name - totožné s jenkins (bez mezer, diakritiky)

module name: nic

Zkoušel jsem ještě deployment key dát přímo do SSH keys v týmu na bitbucket, pořád error fetching...

Nějaký nápady? hejty?
